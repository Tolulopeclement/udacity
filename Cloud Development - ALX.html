What Will You Learn?

Cloud Development

Start by learning the fundamentals of cloud development and deployment with AWS. Then, build different apps leveraging microservices, Kubernetes clusters, and serverless application technology.

Fundamentals of Cloud Computing

Learn the fundamentals of cloud computing while being introduced to computing power, security, storage, networking, messaging, and management services in the cloud.

Modern Cloud Architecture

Modern design and deploying scalable, extendable, and maintainable full-stack applications using modern cloud architecture.

Best Practices

Learn best practices on developing and deploying microservices, focusing on different microservice architecture patterns, independent scaling, resiliency, securing microservices, and best practices for monitoring and logging.

Serverless Technology

Learn both the theory of using serverless technologies with the practice of developing a complex serverless application and focus on learning by doing. You will learn advanced serverless features such as implementing WebSockets and stream processing.
